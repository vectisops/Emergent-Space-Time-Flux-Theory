# Emergent Spacetime Flux Theory (ESFT)

**Independent research exploring a different idea about how the universe expands and forms structure.**

ESFT is **not** a finished product and **not** peer-reviewed gospel. It is early-stage work that asks whether spacetime can be pictured as a kind of energetic “flux” medium — and whether gravity, time, and cosmic expansion can be read as behaviour of that medium — instead of relying only on large amounts of invisible “dark” ingredients.

## What is this, in everyday language?

School and textbook cosmology often says most of the universe is **dark matter** and **dark energy** — labels for things we have not seen the way we see stars and gas. Those labels work well in equations, but they are still placeholders.

ESFT is one independent attempt to explore an alternative picture:

- Imagine spacetime less like a fixed empty stage, and more like a busy medium that can thicken, thin, and flow.  
- **Gravity** might look like “flow” down slopes in that medium.  
- **Time** might relate to how the medium settles and changes.  
- **Expansion** might relate to many tiny motions adding up across the cosmos.  

These are **research metaphors**, not proven facts. The project’s job is to write the ideas clearly, test them against public data where possible, and stay honest about uncertainty.

## Who it is for / typical uses

- Curious readers who want a **non-technical** path into an alternative cosmology idea  
- Independent researchers and students exploring flux / emergent spacetime concepts  
- People building **offline** analysis habits (local compute, local notes)  
- Anyone comparing ESFT’s *questions* with standard dark-sector modelling — without treating either as dogma  

**What it is not:** a theory of everything, an investment pitch, or a claim that observational dark-matter effects “don’t exist.” It is about **modelling frameworks** and early tests.

---

## How the work is approached

1. **State the idea in plain language** (portfolio dossier).  
2. **Write working equations and methods** (technical page + this repo).  
3. **Compare with public astronomical datasets** where practical (for example supernova distance catalogues), using statistical fitting methods.  
4. **Keep claims provisional** until stronger, independent validation exists.  

Early numerical work has suggested the model can track **broad** expansion history in regimes that were checked. That is a starting point, not a final victory. Distinctive behaviour at higher redshift remains an important test.

## Getting started (high level)

```bash
git clone https://github.com/vectisops/Emergent-Space-Time-Flux-Theory.git
cd Emergent-Space-Time-Flux-Theory
```

Windows (math / session path):

```powershell
.\local_enviroment.ps1
```

Linux-style:

```bash
bash local_enviroment.sh
```

Optional local AI analyst (requires [Ollama](https://ollama.com)):

```bash
ollama create esft-analyst -f ESFTModelFile
```

See scripts and `run_session.py` in this repository for the current research workflow.
